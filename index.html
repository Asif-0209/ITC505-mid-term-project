Addendum

In this project, I created an interactive "Game of Thrones" story game that prompts users with different choices and shows images based on their selections. My primary goal was to build a simple yet engaging experience where users can make decisions that affect the story's progression.

For the creative process, I focused on using visual elements (images) to reinforce the storytelling aspect. For example, I utilized images of different Game of Thrones scenes to accompany the choices the user makes, making the experience more immersive.

One of the challenges I faced was ensuring that the images loaded dynamically based on the user's choices. Initially, I had to figure out how to manipulate the DOM (Document Object Model) and dynamically update the story text and images without reloading the page. This required handling JavaScript events to update the content based on user input. I overcame this challenge by using JavaScript to handle click events and display the appropriate story continuation and image.

Another challenge was ensuring that the game reset properly when the user clicked the "Restart Game" button. I had to ensure that the story text, images, and choices all reset to their initial states. This was done by resetting the content of the relevant DOM elements when the reset button was clicked.

Throughout this process, I learned a lot about DOM manipulation and how to create a responsive, interactive web experience. The main insight I gained was the importance of keeping the user experience seamless by ensuring that all the content updates dynamically based on their interactions.
